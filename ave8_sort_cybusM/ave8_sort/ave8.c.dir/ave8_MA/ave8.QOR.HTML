Terminology

(*1) Combinational Loop
See *.err file for detailed loop information.
(*2) Total Net Count
Count one siterminal net is counted as one.
The number of fanouts is not counted.
Nets inside RTL parts such as functional units, multmultiplexers, and decoders,
are included when BLIB version 2.00 or later is usedused.
(*3) Total Pin Pair Count
Count one fanout as 1 for each signal.
Pin pairs inside RTL parts such as functional unitunits, multiplexers, and decoders
are included when BLIB version 2.00 or later is usedused.
(*4) Net Count
Nets inside RTL parts such as functional units, multmultiplexers, and decoders are not included.
(*5) Pin Pair Count
Pin pairs inside RTL parts such as functional unitunits, multiplexers, and decoders are not included.
(*6) Total
The number of all pin pairs counted by every bit for a signal.
(*7) Max
Maximum value among fanouts counted by every bit for for a signal.
The number in ( ) represents the number of bits withwith the maximum fanout.
